HTML, CSS, and JavaScript are the three main building blocks of websites.
HTML is like the skeleton of a webpage. It tells the computer what content to show, like text, images, and links. Think of it as the structure of a house: the walls, windows, and doors.

Click the link to know more information about HTML -> go to site about html

CSS is like the paint and decoration on the walls of that house. It makes the website look nice by adding colors, fonts, and arranging things in a way thats easy to read and navigate. Without CSS, websites would just be plain text and images.

Click the link to know more information about CSS -> go to site about CSS

JavaScript is what makes a website interactive. It allows things to move or change on the page without refreshing it. For example, when you click a button to open a menu or submit a form, JavaScript is working behind the scenes to make that happen.

Click the link to know more information about Javascript -> go to site about Java script


Together, these three tools work to create websites that are not only functional but also attractive and interactive!
