ახსენით რისთვის გამოიყენება შემდეგი ატრიბუტები: type, id, for, name, value
type
ეს ატრიბუტი განსაზღვრავს თუ რა სახის უნდა იყოს input ველი. მაგალითად ეს ატრიბუტი განსაზღვრავს input ველი პროლისთვის იყოს თუ არა, მოსანიშნი უნდა იყოს თუ არა და ასე შემდეგ

id
ეს არის ,,პირადი ნომერი'' თეგებისთვის. ყველა id უნდა იყოს უნიკალური რათა გვერდზე მარტივად გამოვარჩიოთ და დავაკავშიროთ ის სხვა ელემენტებთან. მაგალითად ლინკების დაკავშირება კონკრეტულ ელემენტებთან, მაგალითად, სათაურებთან გადასასვლელად.

for
ეს ატრიბუტი გამოიყენება label თეგში. ამას იყენებენ იმითვის რომ input ველი დააკავშირონ label -თან იმისთვის რომ მომხმარებელმა გაიგოს თუ რა უნდა ჩაწეროს ამ ველში. 

name
ეს ატრიბუტი input თეგებისთვის გამოიყენება. მომხმარებლის მიერ input ველში ჩაწერილი ინფორმაცია ინახება data base-ში. name ატრიბუტი განსაზღვრავს ველის სახელწოდებას ამ data base-ში რომ სერვერმა იცოდეს, რა სახის მონაცემია მიღებული.

value
ესეც input თეგებისთვის გამოიყენება. ეს ატრიბუტი განსაზღვავს ინფორმაციის მნიშვნელობას. აი მაგალითად radio -ებზე შეგვიძლია გამოვიყენოთ. თუ გვინდა, რომ მომხმარებელს მივცეთ არჩევანი (ორი radio) იმისთვის რომ აირჩიოს თუ რა სქესისაა, value ატრიბუტში ვწერთ იმას, რასაც მომხმარებელი აირჩევს. მაგალითი:
